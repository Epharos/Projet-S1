Voilà voilà, j'écris du texte pour faire des tests. Le site se munie de plusieurs outils pour la compréhension : des zones d'information, de questions, d'alertes et d'erreurs. Le code se muni également de sa propre indentation

Et oui, voilà une petite zone d'information !
Le code se décrit comme suit (par contre y a pas de couleurs :/) : 
Code : Python

print "Bonjour,"

prenom = raw_input("Comment t'appelles-tu ? ")

print "Bienvenue", prenom
D'ailleurs, ce code executera : 
Code : Console

> Bonjour,
> Comment t'appelles-tu ? Aurélien
> Bienvenue Aurélien
Attention à bien utiliser 
Code : Python

prenom = raw_input("Comment t'appelles-tu ? ")
Et non pas 
Code : Python

prenom = input("Comment t'appelles-tu ? ")
Sinon l'utilisateur devra utiliser des guillemets pour entrer son prénom ...
Zone d'erreur en guise d'exemple, je n'ai rien à afficher ici pour le moment !
